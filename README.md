# 💣 Bombastic

[![Website](https://img.shields.io/badge/Web_Landing_Page-bombastic--web.vercel.app-red?style=for-the-badge&logo=vercel)](https://bombastic-web.vercel.app/)

> 2~10명이 함께 즐기는 실시간 장기전 소셜 게임 — 폭탄을 돌려라, 마지막에 들고 있는 사람이 진다.

---

## 게임 소개

**Bombastic**은 친구들과 함께 폭탄을 넘기며 벌칙을 피하는 소셜 게임입니다.
그룹을 만들고 초대 링크를 공유해 친구들을 모은 뒤, 방장이 게임을 시작합니다.  
폭탄은 매 **24시간**마다 터지며, 터지는 순간 들고 있던 사람이 폭발 카운트를 받습니다.  
게임 기간(4~7일)이 끝났을 때 폭발 횟수가 가장 많은 사람이 **패배자**가 되어 내기를 이행해야 합니다.

---

## 핵심 규칙

| 항목 | 내용 |
|------|------|
| 인원 | 2 ~ 10명 |
| 게임 기간 | 4 ~ 7일 |
| 폭탄 타이머 | 24시간마다 자동 폭발 |
| 패배 조건 | 게임 종료 시 폭발 횟수 최다인 사람 |
| 전달 순서 | 참여 순서 고정 (아이템으로 변경 가능) |

## 주요 기능

### 🏠 그룹 관리
- **그룹 생성** — 그룹 이름·최대 인원 설정, 6자리 참여코드 자동 발급
- **초대 링크** — `bombastic://join?code=XXXXXX` 딥링크로 앱에서 바로 참여
- **그룹별 닉네임** — 참여할 때마다 해당 그룹에서만 쓸 닉네임 따로 설정
- **강퇴 / 나가기** — 방장은 멤버 강퇴 가능, 일반 멤버는 자진 탈퇴 가능

### 💣 게임 진행
- **실시간 폭탄 현황** — 지금 누가 폭탄을 들고 있는지, 얼마나 남았는지 실시간 표시
- **서버 기반 타이머** — 클라이언트 시간 조작 방지를 위해 서버의 `expiresAt` 기준으로 산정
- **폭탄 전달** — 버튼 한 번으로 다음 사람에게 즉시 패스
- **전달 방향 표시** — 현재 정방향/역방향 시각적으로 표시
- **패스 로그** — 누가 언제 누구에게 넘겼는지 전체 기록

### 🎁 아이템 시스템
랜덤박스(💰 100)를 열어 아이템을 획득하고 전략적으로 사용하세요.

| 아이템 | 효과 | 사용 조건 |
|--------|------|-----------|
| 🔀 순서 뒤섞기 | 멤버 전달 순서를 무작위로 재배치 | 상시 |
| ⏱️ 타이머 단축 | 현재 폭탄의 남은 시간을 대폭 단축 | 폭탄 보유 중 |
| ↩️ 방향 반전 | 폭탄 전달 방향을 반대로 뒤집기 | 상시 |
| 😇 수호천사 | 폭발 시 자동으로 1회 막아줌 | 자동 발동 |

### 💰 재화 & 상점
- **출석 체크** — 매일 앱을 켜고 출석하면 +50💰 (그룹별 독립 적립)
- **미션 완료** — 미션을 달성하면 +30💰
- **랜덤박스** — 💰 100으로 아이템 1개 획득, 확률은 상점 화면에서 확인 가능

### 📊 결과 & 명예의 전당
게임 종료 후 **엔딩 크레딧**이 재생되며 5개 부문 어워드가 수여됩니다.

| 어워드 | 설명 |
|--------|------|
| 💥 패배자 | 폭발 횟수 최다 — 내기를 꼭 이행하세요 😈 |
| 🔥 폭탄 러버 | 폭탄을 가장 오래 들고 있던 사람 |
| 🛡️ 안전제일 | 폭탄을 가장 짧게 들고 있던 사람 |
| 🎯 다재다능 | 아이템을 가장 많이 사용한 사람 |
| 🚀 폭탄 배송 | 폭탄을 가장 많이 넘긴 사람 |

결과 화면에서 **SNS 공유 카드**를 생성해 패배자를 온 세상에 알릴 수 있습니다.

### 🔔 알림
- FCM 푸시 알림으로 폭발·전달 이벤트를 실시간으로 수신

## 화면 흐름

```
로그인 (익명 인증)
  └─ 홈 (내 그룹 목록)
       ├─ 그룹 만들기
       └─ 그룹 참여 (코드 입력 / 딥링크 자동 입력)
            └─ 닉네임 설정
                 └─ 대기실 (초대 링크 공유, 방장: 게임 시작)
                      └─ 게임 중
                           ├─ 홈 탭   — 폭탄 현황, 전달 버튼
                           ├─ 상점 탭 — 랜덤박스 구매, 인벤토리
                           ├─ 미션 탭 — 출석 체크, 미션 목록
                           ├─ 로그 탭 — 패스 기록 전체 조회
                           └─ 설정 탭 — 테마, 그룹 나가기
                                └─ 게임 종료
                                     ├─ 엔딩 크레딧 (명예의 전당)
                                     └─ 결과 페이지 (순위·통계, SNS 공유)
```
